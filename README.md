# Аналоговые и цифровые часы

Небольшое одностраничное приложение-часы, созданное на React и Vite. Приложение отображает адаптивный SVG-циферблат с аналоговыми стрелками и сопровождается цифровым отображением времени и даты. Есть панель настроек и переключатель темы — всё сделано для аккуратного и приятного внешнего вида как на мобильных устройствах, так и на больших экранах.

**Ключевая идея:** лёгкий, наглядный компонент часов с возможностью настроить стиль циферблата и цвет стрелок, а также переключать светлую/тёмную тему.

## Функциональность

- Аналоговый циферблат, выполненный через SVG; плавное движение секундной, минутной и часовой стрелок.
- Цифровое отображение текущего времени (локализация `ru-RU`) и форматированная дата на русском языке.
- Переключение темы: светлая и тёмная тема с соответствующими стилями.
- Панель настроек: выбор стиля цифер (`арабские`, `римские`, `без`) и цветовой схемы стрелок (`классическая`, `синяя`, `золотая`).
- Иконки управления реализованы через `lucide-react`.
- Адаптивный дизайн: интерфейс подстраивается под мобильные и десктопные экраны; на больших экранах реализована двухколоночная раскладка (циферблат слева, настройки/время справа).

## Технологический стек

- React (v19)
- Vite (бандлер и dev-сервер)
- `lucide-react` — набор иконок
- CSS (модуль `src/App.css`) — адаптивная вёрстка и тема

Dev / lint:
- `vite`, `@vitejs/plugin-react`, `eslint` и связанные плагины для разработки и контроля качества кода.

## Структура важных файлов

- `src/App.jsx` — основной компонент приложения, содержит логику времени, отрисовку SVG и панель настроек.
- `src/App.css` — все стили: темы, адаптивная раскладка, оформление циферблата и элементов управления.
